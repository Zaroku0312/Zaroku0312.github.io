Markdown简介，语法

Markdown 介绍

Markdown 是什么？

Markdown是一种轻量级标记语言，创始人为约翰·格鲁伯（英语：John Gruber）。 它允许人们使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML（或者HTML）文档。这种语言吸收了很多在电子邮件中已有的纯文本标记的特性。

出自百度百科 ： https://baike.baidu.com/item/markdown/3245829?fr=aladdin

对我而言，Markdown就是一个方便我写博客的一个工具。

Markdown的基本语法

1.超链接

这个内容用的比较多，我放到前面来说

超链接能做到点击文字跳转网址的效果，插入图片（图片引用）等也要用到超链接

超链接的使用方法：
方括号+小括号 【】（），注意字符都要英文状态

“()”中写入网址即可

而插入图片需要在超链接的基础上加入”！”即可，即！【】（）。 注意字符都要英文状态

“()”中写入图片的地址

插入的图片可以是网络图片和本地图片

本地图片可以放在：hexo博客文件夹对应的“themes”文件夹中“images”文件夹中，可能中间文件夹中的名字会有所不同，例如我的图片所存放的文件夹：G:\bolgword\themes\landscape\source\css\images

方便用于管理，当然你放别的位置也不是不行
